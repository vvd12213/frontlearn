Значением атрибута тега может выступать целое число, содержащее цифры от 0 до 9. Число может быть отрицательным, тогда перед ним ставят знак минус без пробелов (-10).

Пример

<!DOCTYPE html>
<html>
 <head>
  <meta charset="utf-8">
  <title>meter</title>
 </head>
 <body>
  <p>Температура воды</p>
  <meter value="0" max="100" low="10" high="60">Низкая</meter>
  <meter value="30" max="100" low="10" high="60">Нормальная</meter>
  <meter value="80" max="100" low="10" high="60">Горячая</meter>
  <meter value="100" max="100">Кипяток</meter>
 </body>
</html>
