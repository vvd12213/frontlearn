--------------------------------------------------------------------------------------------
1. Состав архива
--------------------------------------------------------------------------------------------

Архив включает в себя следующие папки и файлы (вложенность папок и файлов показана
отступами):

 * <номер урока> - папка с материалами урока с указанным номером. Состав папки:

    - !sources - папка с исходными материалами, необходимыми для выполнения упражнений в
      текущем уроке;

    - ex<номер параграфа> - результаты выполнения упражнения, приведенного в параграфе
      с указанным номером;

    - s<номер параграфа> - результаты выполнения самостоятельных упражнений, приведенных
      в параграфе с указанным номером;

 * I - папка с материалами приложения I. В ней находятся папки с именами вида
   I.<вторая цифра номера параграфа>, которые хранят код элементов, рассмотренных в
   параграфе с указанным номером;

 * II - папка с материалами приложения II. Ее состав такой же, как и у папок с материалами
   уроков (см. ранее);

 * readme.txt - этот текстовый файл.
